# Superpowers Spec Documents

Brainstorm specs and design documents produced by the `superpowers:brainstorming` and `superpowers:writing-plans` skills.

## Naming convention

ISO date prefix + slug: `YYYY-MM-DD-<slug>.md`.

Examples:

- `2026-05-03-claude-project-setup-design.md`
- `2026-05-15-add-options-greeks-agent.md`

## Convention

Each spec contains:

1. Goal and non-goals.
2. Per-artifact design (file paths, content sketches).
3. Self-review (placeholders, contradictions, ambiguity).
4. Implementation order.
5. Verification checklist.
6. Out-of-scope deferred work.
